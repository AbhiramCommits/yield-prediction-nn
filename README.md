# Yield Prediction using Neural Networks

A machine learning pipeline for predicting semiconductor deposition yield from process parameters using PyTorch neural networks.

## Motivation

Semiconductor manufacturing yield is a critical metric directly impacting fab profitability. Traditional statistical process control methods offer limited predictive power when process variables interact nonlinearly. This project explores whether feedforward neural networks can match or exceed the performance of conventional regression approaches and random forest ensembles on a synthetic yield prediction task, with the goal of establishing a reproducible benchmark for ML-driven process optimization.

## Dataset

The dataset (`data/yield_data.csv`) contains **8,000 synthetic samples** simulating a plasma-enhanced chemical vapor deposition (PECVD) process. Six process features are used to predict a continuous yield percentage.

| Feature | Description | Range | Unit |
|---|---|---|---|
| `temperature_C` | Deposition temperature | 200 – 400 | °C |
| `pressure_mTorr` | Chamber pressure | 10 – 500 | mTorr |
| `gas_flow_sccm` | Gas flow rate | 50 – 500 | sccm |
| `rf_power_W` | RF power | 100 – 1,000 | W |
| `deposition_time_s` | Deposition duration | 60 – 600 | s |
| `chamber_humidity` | Relative chamber humidity | 20 – 80 | % |
| `yield_percent` | **Target** — process yield | 60 – 99 | % |

The target is generated by a polynomial function of the features with additive Gaussian noise (σ = 3.5), producing realistic nonlinear relationships and interaction effects.

## Model Architecture

```
Input (6)
  |
  v
Linear(6, 128)
ReLU
Dropout(0.1)
  |
  v
Linear(128, 64)
ReLU
Dropout(0.1)
  |
  v
Linear(64, 32)
ReLU
Dropout(0.1)
  |
  v
Linear(32, 1)  -->  Yield (%)
```

- **Parameters:** ~10,000 trainable
- **Activation:** ReLU throughout
- **Regularization:** Dropout (p = 0.1) after each hidden layer
- **Loss:** Mean Squared Error
- **Optimizer:** Adam (lr = 0.001)
- **Batch size:** 64
- **Epochs:** 50 (tuned)

## Results

| Model | RMSE | R² |
|---|---|---|
| Linear Regression | 3.83 | 0.839 |
| Ridge Regression | 3.83 | 0.839 |
| Random Forest | 3.58 | 0.859 |
| **Neural Network (ours)** | **3.56** | **0.861** |
| Chen et al. (2023) — benchmark | — | 0.84 |

The tuned neural network achieves the best performance across all metrics, surpassing the random forest baseline and the literature benchmark. Deposition time is the most important predictor by permutation importance (R² decrease ≈ 0.88), followed by chamber pressure and temperature.

## How to Reproduce

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Generate the synthetic dataset

```bash
python src/generate_data.py
```

### 3. Run baseline models

```bash
python src/baseline_models.py
```

### 4. Run hyperparameter tuning (trains best model)

```bash
python src/hyperparam_tuning.py
```

### 5. Explore notebooks

```bash
jupyter notebook notebooks/
```

| Notebook | Description |
|---|---|
| `01_eda.ipynb` | Exploratory data analysis — distributions, correlations, pairplots |
| `02_results_analysis.ipynb` | Loss curves, predictions, permutation importance |
| `03_paper_reproduction.ipynb` | Structured academic notebook with full methodology |
